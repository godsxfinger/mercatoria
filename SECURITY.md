# Security Policy

## Reporting Security Vulnerabilities

I take security vulnerabilities seriously and appreciate responsible disclosure.

**Please do not report security vulnerabilities through public GitHub issues.**

## How to Report

- Email: `security@example.com`
- Subject: `[SECURITY] Vulnerability Report - Mercatoria`
- Public-key encryption is preferred but not mandatory

## Public Key

The current public key is published at `storage/app/public/pgp_key.txt` and exposed by the `/pgp-key` route.

## What to Include

- Description: a short description of the vulnerability
- Steps to reproduce: detailed steps to reproduce the issue
- Suggested fix: an implementation idea if you have one
- Contact information: how I can reach you for follow-up

## Acknowledgments

Security researchers who contribute to Mercatoria through responsible reports or security-related pull requests can be acknowledged here if they wish.
